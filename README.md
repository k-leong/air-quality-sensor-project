# Air Quality Sensor Project using SparkFun Environmental Combo Breakout  

## BOM
- SJ One Board
- SparkFun Environmental Combo Breakout - ENS160/BME280 (Qwiic)
---
Setup was done using the [SJSU Dev guide](https://sjsu-dev.readthedocs.io/en/latest/?badge=latest) for initial SJ One Board setup and [SJSU-Dev-Linux Version](https://github.com/kammce/SJSU-DEV-Linux) for the telemetry setup

All drivers I have written are under L2_Drivers_v2 directory
